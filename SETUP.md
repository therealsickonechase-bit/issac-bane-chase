# Repository Setup

## Related Repository

This project references the Bane_Issac repository.

### Adding the Related Repository as a Remote

To add the related repository as a git remote for reference:

```bash
git remote add bane-issac https://github.com/AiG3/Bane_Issac.git
```

### Viewing Remotes

To view all configured remotes:

```bash
git remote -v
```

This will allow you to fetch information from the related repository if needed.
