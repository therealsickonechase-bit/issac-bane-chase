# Isaac Bane Chase: Path to Freedom

A narrative adventure game about breaking free from medical and institutional control.

## Game Concept

Isaac Bane is trapped in a medical facility under CIA surveillance. The game explores themes of:
- Personal autonomy and freedom
- Overcoming institutional control
- Mental and physical liberation
- Trust and deception

## Development Status

This is an Adventure Game Studio (AGS) game project currently in early development.

## Game Objective

Guide Isaac through the facility, making choices that will determine his path to freedom. Each decision affects trust, resources, and available escape routes.

## Core Mechanics

- **Freedom Meter**: Tracks progress toward liberation
- **Trust System**: Build relationships with allies
- **Resource Management**: Gather tools and information
- **Stealth & Strategy**: Avoid detection while planning escape
- **Multiple Endings**: Different paths lead to different outcomes

## Development Roadmap

1. Story and character development
2. Game design documentation
3. Asset creation (sprites, backgrounds, audio)
4. AGS implementation
5. Testing and refinement

## Getting Started

This project uses Adventure Game Studio. To work on this game:
1. Install AGS Editor
2. Clone this repository
3. Open the project in AGS Editor

## Theme

The central theme is freedom - breaking physical, mental, and institutional chains to reclaim autonomy and self-determination.
